# Отчет: Матричное умножение на OpenCL

## Конфигурация

- Платформа: Apple
- Устройство: Apple M4 GPU
- Размеры матриц: A[512 x 512] * B[512 x 512] = C[512 x 512]
- Рабочая группа: 16 x 16

## Результаты

| Метод | Время (сек) | Ускорение |
|-------|-------------|-----------|
| CPU (последовательно) | 0.383079 | 1.00x |
| GPU (только ядро) | 0.007565 | 50.64x |
| GPU (с передачей данных) | 0.008022 | 47.75x |

## Выводы

1. GPU показывает ускорение ~50x для матричного умножения
2. Накладные расходы на передачу данных минимальны (~6%)
3. Результаты GPU полностью совпадают с CPU
4. Матричное умножение хорошо параллелизуется на GPU
