# Отчет: Сравнение производительности OpenCL на CPU и GPU

## Конфигурация

- Платформа: Apple
- Устройство: Apple M4 GPU
- Размер массива: 16 777 216 элементов (64 MB)
- Операция: Сложение двух массивов (C = A + B)

## Результаты измерений

| Метод | Время (сек) | Ускорение |
|-------|-------------|-----------|
| CPU (последовательно) | 0.025374 | 1.00x |
| OpenCL GPU (только ядро) | 0.018462 | 1.37x |
| OpenCL GPU (ядро + чтение) | 0.021832 | 1.16x |

## Выводы

1. OpenCL на GPU показывает ускорение 1.37x для вычислений
2. С учетом передачи данных ускорение составляет 1.16x
3. Накладные расходы на передачу данных значительны для простых операций
4. Для более сложных вычислений преимущество GPU будет выше
